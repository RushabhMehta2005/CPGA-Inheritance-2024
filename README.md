## 🛠Project Setup

For the Web-App 1.Clone the GitHub repo.
```bash
git clone https://github.com/RushabhMehta2005/CPGA-Inheritance-2024/
```
2.Enter the client directory. Install all the required dependencies.
```bash
  cd frontend
  npm install
  npm start
```

3.To start the backend server:
```bash
  cd backend
  npm install
  nodemon server.js
```
